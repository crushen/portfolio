What can I do for you?

Designing and developing visual aspects of the web is my thing,
writing every line of code with responsiveness and creativity in mind.

HTML & CSS

Modern and semantic

Using these two core technologies, I can bring any website design to life - ensuring that semantic HTML is used to properly structure the page. When it comes to styling I love all things CSS, from flexbox to animation, and I am able to use modern libraries like Bootstrap and Materialize.

JavaScript

Adding functionality

There's nothing more satisfying than taking a static web page, and making it interactive with JavaScript. I'm a natural problem solver, which is great for turning complex ideas into something fully functioning. I mainly use vanilla JavaScript, but I'm also familiar with Vue.js and jQuery.

Responsive Design

From mobile to desktop

Most internet users access the web by a mobile device, so it's important that websites are designed using responsive design principles. I will always aim to put the users experience before anything else, and often design with a mobile first approach.
